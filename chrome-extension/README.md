# Pomodoro Chrome Extension

Extension for Google Chrome - Pomodoro, breaks, breaks durations and App Theme can be edited, data saved in Chrome Storage.

## Build

npm run build
yarn build

npm run dev - for debugging
yarn dev

## Chrome extension

- HTML/SCSS
- TypeScript/JavaScript
- Webpack
- Chrome Storage
