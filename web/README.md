# Pomodoro Web Application

Pomodoro Web App - Pomodoro, breaks, breaks durations and App Theme can be edited, data saved in Local Storage.

## Web Application

- React
- Redux Toolkit
- TypeScript
- Webpack

## Build

npm run build
yarn build

npm run dev - for debugging
yarn dev
